# Project Management Web App

## Overview
The Project Management & Timesheet Web App is a web-based application designed to help teams manage projects, tasks, and employee work efficiently.

The system allows users to create projects, manage tasks, track work duration, and maintain timesheets. It includes authentication and role-based access where admins can manage employees and projects, while employees can manage their tasks and log working hours.


## Features

### Authentication
- User Sign Up
- User Login
- Secure authentication system
- Profile update option
- Password change functionality


### Admin Features
Admin users have additional permissions such as:

- Manage employee accounts
- Edit employee information
- Monitor project activities
- Control project visibility
- Manage projects and tasks


### Employee Features
Employees can:

- Log in to the system
- View assigned projects
- Edit their profile
- Change their password
- Log working hours using the timesheet system


### Project Management
Users can manage projects with the following options:

- Create projects
- Edit project details
- Delete projects
- Set project visibility

Project Visibility Types:
- Public Project → Visible to all users
- Private Project → Visible only to selected users


### Task Management
Inside each project users can:

- Create tasks
- View task details
- Track task duration
- Manage project progress


### Timesheet System
The application includes a timesheet tracking system where employees can:

- Log time spent on tasks
- Track working hours
- View monthly work summaries
- Manage daily time logs

## Installation & Setup

1. Clone the repository

git clone https://github.com/yashasvipatel03/project-management-web-app

2. Navigate to the project folder

cd project-management-web-app

3. Install dependencies

npm install

4. Run the application

npm start

5. Open the application in browser

http://localhost:3000
